# Contributing to Loom

We welcome contributions! Here's how you can help:

## Getting Started
1. Clone the repo: `git clone https://github.com/your-org/loom`
2. Install dev dependencies: `pip install -e ".[full]"`

## Coding Style
- Follow PEP 8
- Use `black` for formatting
- Add type hints where possible

## Testing
- Run `pytest` from the root directory
- Test on a small repo: `loom init && loom plan "fix typo"`

## Feature Requests
Open an issue with the label `enhancement` and describe your use case.
